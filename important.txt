1. Component Types

No official Differentiation but normally
Any frame work or library has the component type.

  1) Containers
    1) Stateful components
    2) Hold & manage data, pass data around
    3) <App />, <Products />, <Checkout />...
    4) Only a few (depending on app size per app)
    

  2) Components
    1) Presentational components
    2) Receive & output data (which means rendering data in the browser.)
    3) <Modal />, <Card />, <Button />
    4) Use many!
